# Badges - Значки ролей

Здесь должны находиться значки ролей для профилей:

Каждый файл должен называться по ID роли, например:
- `badge_role_1.png` - Значок для роли с ID badge_role_1
- `badge_role_2.png` - Значок для роли с ID badge_role_2

## Примечание:
Переместите сюда все PNG файлы значков из папки badges в корне проекта.
